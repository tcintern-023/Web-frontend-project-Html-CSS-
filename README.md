# Personal Portfolio Website

A clean and modern personal portfolio website created to showcase my skills, projects, and background as a Computer Science student and aspiring Web Developer.

## 📌 About The Project

This project is a personal portfolio landing page built using HTML and CSS. It presents my introduction, technical skills, projects, and contact information in a simple and responsive design.

The goal of this project was to practice and improve my frontend development fundamentals, especially HTML structure, CSS styling, Flexbox layouts, and responsive design.

## ✨ Features

- Hero section with personal introduction
- Professional profile image
- About Me section
- Skills section
- Projects showcase
- Contact section with form
- Smooth navigation between sections
- Clean and modern UI design
- Responsive layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- Google Fonts
